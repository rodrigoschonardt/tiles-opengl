# Aplicação de movimentação por tiles

Um projeto utilizando OpenGL para renderização de mapas de tiles (tilemaps) com interação via teclado.

## Funcionalidades
- Renderização de um mapa de tiles 3x3.
- Carregamento de spritesheets usando `stb_image`.
- Movimentação de cursor (W, A, S, D).
- Alteração dinâmica de tiles (Barra de Espaço).
